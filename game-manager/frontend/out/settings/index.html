Settings
Layout matches the game client shell: header, sidebar (categories + quick access), main grid, footer. Changes save automatically.
Launch on Windows startup
Adds game-client.exe (same folder as Game Manager) to Windows startup. The switch reloads from HKCU\…\Run (EZJRGameClient).

Popular list file (diskless / network)

Optional absolute path to popular.json, or a folder that contains it (e.g. \\fileserver\share\launcher\popular.json). Empty = use data/popular.json next to the apps. Format: { "gameIds": [1,2,3] }. The client also writes launch-stats.json in the same folder (open counts per game id).

Quick Access shortcuts

Pin and reorder games shown in the client launcher quick access strip.

Background images are optional. If missing, the launcher will fall back to the theme dark color.
